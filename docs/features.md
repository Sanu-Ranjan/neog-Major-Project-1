# GrillMart

> E-commerce app for baking,
> grill & tandoor equipment

---

## Features

### Home Page

- Landing page with featured
  category cards
- Clicking a category goes to
  the product listing page with
  that category pre-selected

### Product Listing Page

- All products listed with a
  filters sidebar
- Each card shows: image, name,
  price, rating, add to cart,
  add to wishlist

### Filters

- **Category** : checkbox filter
- **Ratings** : slider filter
- Clear Filters button to reset
  all applied filters

### Sort by Price

- Radio button to sort products
  low to high or high to low

### Product Detail Page

- Click any product to see its
  full details
- Add to Cart and Add to
  Wishlist buttons on this page

### Wishlist Management

- View all wishlisted products
  from the navbar
- Remove item from wishlist
- Add item to cart (increases
  qty if already in cart)

### Cart Management

- View all cart products from
  the navbar
- See and change quantity
- Remove product from cart
- Move product to wishlist
- Price details card with
  checkout button

### Address Management

- Add, update and delete
  multiple addresses
- Choose one address to deliver

### Checkout

- Choose address and place order
- Shows order summary and
  "Order Placed Successfully"
- Orders saved to the backend

### User Profile

- Shows: name, email, phone,
  address
- Option to add a new address
- Option to view order history

### Loading & Alerts

- Loading message while
  products are fetching
- Toast alerts for all
  cart & wishlist actions

### Search

- Search products by name
  via the navbar search bar
